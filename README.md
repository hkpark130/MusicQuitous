# MusicQuitous
<h2>概要</h2>
オンライン上で合奏ができる環境を提供するサービスです。<br>
<h2>特徴</h2>
●　ユーザが録音した歌に他のユーザーが自分の演奏を録音して載せる形式の合奏<br>
●　歌を構成する楽器リストの中で自分が望む楽器を選んで錄音する形式<br>
●　ユーザがよく演奏した歌や楽器を分析して他のユーザとのマッチング<br>
